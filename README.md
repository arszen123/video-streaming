# Simple video streaming site

## Features

- Upload video in chunks
- Stream video in chunks

## Setup (GCP local)
1. Create new GCP.
2. Open Cloud shell.
3. Install ```node```, ```npm```.
4. Clone repository.
4. Run ```./setup_environment.sh```
5. Run ```npm install```
6. Run ```npm start```
6. Click "Web preview" > "Preview on port 8080"
